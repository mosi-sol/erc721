## NFT + Royality
- erc721 included:
- - bulk mint
- - royality
- - air drop
- included flat version

### how to deploy & info
- MAXSUPPLY = maximum items mintable
- PRICE = per item (in wei)
- ROYALITY = as a percntage -> 100000 = 100% -> 10000 = 10% 
- URIPREFIX = hash code from IPFS -> Cid
